📊 Project Case Studies

Real-world challenges solved through strategic QA approach

Authentication Testing — GuestApp

🎯 Challenge

Authentication is one of the most critical areas in any application. Built a dedicated app from scratch to serve as a controlled environment for both manual and automation testing.

💡 Solution

Designed manual test cases in spreadsheet, then implemented automation using Playwright for E2E flows and Vitest + Supertest for API-level testing. Documented entire testing process from planning to bug resolution.

📈 Impact

↓ Playwright ↓ Vitest + Supertest ↑ Manual Test Case

Payment & Billing QA — Alfamart Virtual Business

🎯 Challenge

Testing payment and billing features in a large-scale retail e-commerce environment, where financial accuracy directly impacts end customers and enterprise operations.

💡 Solution

Conducted functional and UAT testing across payment modules including electricity and PDAM billing. Documented bugs with structured reports covering description, expectation, priority, and developer handoff notes.

📈 Impact

↓ Functional Testing ↑ UAT Bug Reporting

Geofencing Validation — Promotion Media App

🎯 Challenge

Developed a location-based promotion application using geofencing and the Crossing Number algorithm. Required rigorous validation to ensure promotions only triggered within the correct geographic areas.

💡 Solution

Performed functional validation by testing promotion triggers across defined geographic zones. Verified that the Crossing Number calculation correctly determined whether a user was inside or outside a target area.

📈 Impact

↑ Flutter + Firebase ↓ Functional Validation ⚡ Algorithm Testing
